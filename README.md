# AI Manga Drama Lab / AI漫剧制作实验室

从剧本、角色、分镜、Prompt、图像生成、视频生成、配音、剪辑、发布、复盘到商业化，系统学习 AI 漫剧制作全流程。

**这不是工具清单，而是一套可执行的 AI 漫剧生产方法和商业化学习系统。**

## 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS
- Framer Motion
- lucide-react
- Recharts

## 安装与运行

```bash
npm install
npm run dev      # 启动开发服务器
npm run build    # 构建生产版本
npm run preview  # 预览构建产物
```

## 部署

项目是纯前端静态站，构建产物在 `dist/`。可部署到：

- **Vercel**: 导入 Git 仓库，自动检测 Vite 配置
- **Netlify**: 构建命令 `npm run build`，发布目录 `dist`
- **GitHub Pages**: 使用 `gh-pages` 或 GitHub Actions

| 平台商业化对比 | 8大平台的变现方式与门槛 |

## 项目结构

```
src/
  components/         # React 组件 (20个)
  data/               # 数据文件 (12个)
  hooks/              # 自定义 Hook
  lib/                # 工具函数
  App.tsx             # 主应用
  main.tsx            # 入口
  index.css           # 全局样式
```

## 数据更新机制

- **工具价格**: 编辑 `src/data/tools.ts`，更新 `pricing`、`lastChecked` 字段
- **平台政策**: 编辑 `src/data/platforms.ts`，更新对应字段
- **来源索引**: 编辑 `src/data/sources.ts`，同步更新 `SOURCES.md`
- 所有价格、政策、分账、商用授权字段都保留 `lastChecked` 或标注"需实时确认"
